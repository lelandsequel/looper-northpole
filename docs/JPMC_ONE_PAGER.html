> THE SCORING ALGORITHM (100% DETERMINISTIC)

The priority score is a modified RICE framework where "Impact" is the log-scaled 3-Year Net Present Value (NPV).

1. RICE Score: Score = (Reach * Impact * Delivery_Confidence) / Effort

- Impact: ln(1 + 3_Year_NPV)

2. 3-Year NPV: Σ [ (Net_Cash_Flow_t) / (1 + 0.10)^t ] for years t = 1 to 3

- Net Cash Flow: (Revenue * (1-COGS)) + Cost_Savings + (Cost_Avoidance * pAvoid) + ...

All financial inputs to the NPV calculation are required to have a cited source to pass the CADMUS intake gate.
