> THE ALGORITHM (100% DETERMINISTIC)

1. RICE Score: Score = (Reach * Impact * Delivery_Confidence) / Effort

- Impact: ln(1 + 3_Year_NPV)

2. 3-Year NPV: Σ [ (Net_Cash_Flow_t) / (1 + 0.10)^t ] for years t = 1 to 3

- Net Cash Flow: (Revenue * (1-COGS)) + Cost_Savings + (Cost_Avoidance * pAvoid) + (Risk_Reduction * pRisk) + Customer_Impact - Ongoing_TCO
